# LICENSES Directory

This folder collects the license texts that govern this release:

- `CC-BY-4.0.txt` — Legal code for Creative Commons Attribution 4.0 International (applies to our original
  CEFR annotations, labels, and dataset splits).
- `WORDNET-3.0-LICENSE.txt` — Princeton WordNet 3.0 license text (required because we include WordNet
  sense identifiers (sense keys) and/or redistribute files derived from WordNet 3.0).
- `SEMCOR-3.0-LICENSE.txt` — The SemCor 3.0 distribution uses the same Princeton license terms.
  Include this text when redistributing SemCor-derived content (e.g., SemCor sentences or IDs).

## How these licenses stack

Our annotations are provided under CC BY 4.0. Files that include or derive from Princeton data must **also**
include the Princeton license texts. Keep both sets of terms with any redistribution.
